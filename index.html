Backup & Restore

📱 How to back up your phone progress:
1. Tap Backup Now — a file downloads
2. On iPhone: Files app → tap file → Share → email it to yourself
3. On computer: open the email, save the file
4. Open VietCards on computer → tap Restore from File → pick the file

Tip: Back up after every study session so you never lose progress!

Google Sheets Sync

Not configured.
Auto-sync on app open
Setup (one time):
1. In your Google Sheet, name each tab to match a deck name (case-insensitive).
2. Each tab needs columns: English | Vietnamese | Breakdown | Note (in row 1).
3. File → Share → "Anyone with the link" → Viewer.
4. Paste the sheet URL above and tap Sync Now.

What syncs: Card content (English, Vietnamese, breakdown, notes).
What's preserved: Your study progress (box level, due dates, starred, marked words).
What's NOT auto-deleted: Cards that exist locally but not in the sheet stay (delete them manually if needed).
